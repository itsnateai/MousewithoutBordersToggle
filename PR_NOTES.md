## Findings by Priority

### P1 (Important) — Fixed
- **FileRead crash on locked file** — `DoToggle()`, `SyncTray()`, `PauseSharing()`, and `ResumeSharing()` all called `FileRead()` without try/catch. If MWB locks `settings.json` during a read, AHK throws an unhandled exception crashing the script. Added try/catch with graceful fallback to all 4 call sites.
- **Invalid hotkey from INI crashes startup** — `Hotkey(g_hotkey, ...)` at script init would throw if `MWBToggle.ini` contained an invalid hotkey string. Added try/catch with user warning and fallback to default `Ctrl+Alt+C`.

### P2 (Moderate) — Fixed
- **Explorer restart loses tray icon** — No `TaskbarCreated` message handler. If Explorer crashes/restarts, the tray icon disappears permanently. Added `OnMessage(WM_TASKBARCREATED, ...)` to call `SyncTray()` on Explorer restart.
- **README screenshot placeholder visible to users** — Line "Replace this section with real screenshots when available" was user-facing. Removed.

### P3 — Fixed
- **README missing INI config docs** — Customization section only mentioned editing the script directly. Added full table documenting `MWBToggle.ini` keys (Hotkey, ConfirmToggle, SoundFeedback) with defaults and descriptions.

### P4 — Fixed
- **MWBToggle.ini not in .gitignore** — User config file could be accidentally committed.

### P3-P4 — Logged in AUDIT_TASKS.md (local, gitignored)
